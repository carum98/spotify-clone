# <img src="https://nuxt.com/icon.png" width="30"> 

**Spotify clone** build with Nuxt 3

## SetUp
This project use [Spotify API](https://developer.spotify.com/)

Create ``.env``:

```
NUXT_CLIENT_ID=[CLIENT_ID]
NUXT_CLIENT_SECRET=[CLIENT_SECRET]
```
